### 1.1 策略定义

**目标**: 稳健收益，低风险
**股票池**: 全市场市值TOP 200 + 成分股
**核心因子**:
- 估值因子: EV_EBITDA, PE_TTM, PB, PS
- 质量因子: ROE, ROIC, 利润率
- 盈利因子: 营收增长、利润增长

**选股逻辑**:
1. 流动性过滤: 日成交额 > 5000万
2. 质量过滤: ROE > 12%, 负债率 < 70%
3. 价值过滤: EV_EBITDA < 行业中位数
4. 综合评分Top 15

**目标仓位**: 35%

#### 策略B：高成长策略（新增）
**目标**: 捕捉高β收益
**股票池**: 创业板 + 科创板 + 中证500成分股
**核心因子**:
- 成长因子: 营收增长CAGR(3年)，利润增长CAGR(3年)
- 创新因子: 研发费用占营收比，专利申请数（可选）
- 动量因子: 20日动量，相对强度

**选股逻辑**:
1. 成长性过滤: 营收3年CAGR > 20% 或 利润3年CAGR > 25%
2. 动量过滤: 20日动量 > 中位数
3. 质量过滤: ROE > 8%，避免纯概念股
4. 综合评分Top 10

**目标仓位**: 25%

**风险控制**:
- 单策略最大回撤: -18%
- 持股集中度: 单股 < 8%

**目标**: 寻找被低估的错杀股
**股票池**: 全市场（排除ST、长期停牌）
**核心因子**:
- 估值因子: PB < 1.5, PE < 10, EV/EBITDA < 5

**选股逻辑**:
1. 价值过滤: PB < 1.5 或 PE < 10
2. 财务健康: 资产负债率 < 60%，经营现金流 > 0
3. 反转信号: RSI < 30 或 近60日跌幅 > 15%
4. 综合评分Top 10

**目标仓位**: 20%

**风险控制**:
- 价值陷阱检测: 负增长率 > -20%时权重减半
- 止损线: 单股-15%止损

**目标**: 捕捉趋势行情
**股票池**: 全市场活跃股（换手率 > 2%）

**选股逻辑**:

4. 综合评分Top 15

**目标仓位**: 20%

**风险控制**:
- 趋势反转检测: MA5下穿MA20时减半

## 第二部分：新因子开发计划

### 2.1 另类数据因子（Phase 1, 优先级：高）

#### 2.1.1 新闻情感因子
**数据源**: 东方财富新闻、新浪财经、券商研报
**计算方法**:
```
新闻情感得分 = Σ(情感指数 × 文章权重)
情感指数 ∈ [-1, -0.5, 0, 0.5, 1]  (负面到正面)
文章权重 = 基于发布时间、来源权威度、阅读量
```

**实现步骤**:
1. 接入新闻API（tushare新闻接口）
2. 使用情感分析模型（SnowNLP或BERT中文）
3. 按股票聚合7日情感得分
4. 计算周度变化率作为因子值

**数据特征**:
- 更新频率: 每日
- 覆盖度: 全市场
- 预期IC: 0.05-0.08

---

#### 2.1.2 社交媒体情绪因子
**数据源**: 微博/雪球/股吧帖子（模拟或真实）
**计算方法**:
```
社交媒体情绪 = 正面帖子数 - 负面帖子数 / 总帖子数
情绪变化率 = (今日情绪 - 昨日情绪) / 昨日情绪
热度加权 = 帖子数量 × 评论数 × 点赞数
```

**实现步骤**:
1. 爬取雪球/股吧帖子（需模拟或API）
2. 文本情感分析（同上）
3. 异常检测（过滤水军、广告）
4. 构建情绪指数

**数据特征**:
- 更新频率: 每日
- 覆盖度: 优先创业板、科创板
- 预期IC: 0.03-0.06（高噪音因子）

---

#### 2.1.3 搜索热度因子
**数据源**: 百度指数、微信指数（模拟数据或替代指标）
**计算方法**:
```
搜索热度得分 = (当前搜索热度 - 30日均值) / 30日标准差
关注度排名 = 搜索热度在市场中的百分位
```

**实现步骤**:
1. 采集搜索热度数据（可使用网络爬虫）
2. 标准化处理
3. 滚动计算Z-score
4. 异常值处理（剔除暴涨暴跌）

**数据特征**:
- 更新频率: 每周
- 覆盖度: 知名度较高股票（前300只）
- 预期IC: 0.02-0.05

---

### 2.2 高频因子（Phase 1, 优先级：高）

#### 2.2.1 日内动量因子
**数据源**: 分钟级K线数据
**计算方法**:
```
上午动量 = (14:00收盘价 - 09:30开盘价) / 09:30开盘价
下午动量 = (15:00收盘价 - 13:00开盘价) / 13:00开盘价
日内强度 = 上午动量 × 0.4 + 下午动量 × 0.6
```

**实现步骤**:
1. 获取分钟数据（tushare、baostock）
2. 计算分时动量
3. 滚动5日均值作为因子值

**数据特征**:
- 更新频率: 每日
- 覆盖度: 全市场
- 预期IC: 0.04-0.07

---

#### 2.2.2 成交量异常因子
**数据源**: 日度成交额、成交量
**计算方法**:
```
成交额异常 = (今日成交额 - 20日均值) / 20日标准差
量价背离 = 成交额涨幅 × (1 - 股价涨幅)
```

**实现步骤**:
1. 计算20日滚动均值和标准差
2. Z-score标准化
3. 捕捉异常点（>2或<-2）

**数据特征**:
- 更新频率: 每日
- 覆盖度: 全市场
- 预期IC: 0.03-0.05

---

#### 2.2.3 资金流向因子
**数据源**: 逐笔数据或大单追踪（可选，或使用间接指标）
**计算方法**:
```
净流入估算 = 成交额 × (收盘价-开盘价)/高价-低价
主力估算 = 大单成交额 - 大单卖出额
```

**实现步骤**:
1. 基于高低价和收盘价估算资金流向
2. 滚动计算5日累计流向
3. 标准化作为因子值

**数据特征**:
- 更新频率: 每日
- 覆盖度: 全市场
- 预期IC: 0.02-0.04

## 第三部分：组合优化方案

### 3.1 风险平价（Risk Parity）模块

#### 3.1.1 核心思想
等风险贡献原则，而非等权重。根据每个策略的风险水平动态分配仓位。

#### 3.1.2 实现方法
**步骤1: 计算各策略波动率**
```python
# 使用历史60日收益率计算波动率
vol_A = std(returns_A_60d)
vol_B = std(returns_B_60d)
vol_C = std(returns_C_60d)
vol_D = std(returns_D_60d)
```

**步骤2: 计算目标权重**
```python
# 风险平价权重 ∝ 1/波动率
weight_A = 1 / vol_A
weight_B = 1 / vol_B
weight_C = 1 / vol_C
weight_D = 1 / vol_D

# 归一化
total_weight = weight_A + weight_B + weight_C + weight_D
weight_A /= total_weight
# ...
```

**步骤3: 应用约束**
```python
# 上下限约束
weight_A = np.clip(weight_A, 0.25, 0.45)  # 蓝筹策略范围25%-45%
weight_B = np.clip(weight_B, 0.15, 0.35)
weight_C = np.clip(weight_C, 0.10, 0.30)
weight_D = np.clip(weight_D, 0.10, 0.30)
```

**步骤4: 相关性调整**
```python
# 如果策略间相关性过高，降低权重
if corr(A, B) > 0.7:
    weight_A *= 0.8
    weight_B *= 0.8
```

#### 3.1.3 调整频率
- **月度调整**: 每月1日根据上月波动率调整权重
- **周度微调**: 若某策略波动率变化超过±30%，触发周度调整
- **紧急调整**: 单策略单日跌幅>8%时，临时降低其权重至70%

### 3.2 动态权重调整

#### 3.2.1 基于市场环境的权重调整

**牛市环境识别**:
```
指标: 中证全指 > 200日均线 + MACD金叉
权重调整: 动量策略↑5%，价值策略↓5%
```

**熊市环境识别**:
```
指标: 中证全指 < 200日均线 + MACD死叉
权重调整: 蓝筹策略↑5%，动量策略↓5%
```

**震荡市识别**:
```
指标: 中证全指在±5%区间震荡
权重调整: 等权配置
```

#### 3.2.2 基于策略表现的权重调整

**策略表现奖励**:
```python
# 连续2月正收益且跑赢基准
if strategy_returns_A > benchmark_returns and consecutive_months > 2:
    weight_A += 0.02  # +2%
```

**策略表现惩罚**:
```python
# 连续2月负收益且跑输基准
if strategy_returns_B < benchmark_returns and consecutive_months > 2:
    weight_B -= 0.02  # -2%
```

**权重调整限制**:
- 单次调整幅度: ±2%
- 累计调整幅度: ±5%（相较于初始权重）

### 3.3 自适应调整系统

#### 3.3.1 在线学习权重（可选，高级）

使用自适应算法动态学习最优权重分配：

**算法选择**:
- UCB (Upper Confidence Bound)
- Thompson Sampling
- 强化学习（RL）

**实现框架**:
```
状态: S(t) = [市场环境, 各策略历史收益]
动作: A(t) = [权重A, 权重B, 权重C, 权重D]
奖励: R(t) = 组合夏普比率
目标: 最大化 Σ(R(t))
```

#### 3.3.2 风险预算控制

**最大回撤控制**:
```python
# 当组合回撤接近警戒线时，降低所有策略仓位
if portfolio_drawdown > -10%:
    # 将所有策略仓位乘以0.85（降低至85%）
    weights *= 0.85

if portfolio_drawdown > -12%:
    # 临时切换至现金仓50%
    weights *= 0.5
```

**波动率控制**:
```python
# 当组合波动率超过目标波动率时，调低仓位
if portfolio_volatility > target_volatility:
    scale_factor = target_volatility / portfolio_volatility
    weights *= scale_factor
```

#### 4.2.1 策略A - 蓝筹价值策略

**实现步骤**:
1. 股票池筛选: 全市场市值TOP 200 + 指数成分股

**关键代码**:
```python
def bluechip_value_strategy(data, n_stocks=15):
    # 1. 股票池筛选
    pool = data[data['market_cap'] > data['market_cap'].quantile(0.9)]

    # 2. 因子标准化
    ev_ebitda_norm = normalize(pool['EV_EBITDA'], invert=True)  # 低值为好
    pe_norm = normalize(pool['PE_TTM'], invert=True)
    roe_norm = normalize(pool['ROE'])  # 高值为好

    # 3. 综合得分
    pool['score'] = ev_ebitda_norm * 0.4 + pe_norm * 0.3 + roe_norm * 0.3

    # 4. 选股
    selected = pool.nlargest(n_stocks, 'score')

    return selected
```

def high_growth_strategy(data, n_stocks=10):
    # 1. 股票池筛选
    pool = data[data['market'] in ['创业板', '科创板']]

    # 2. 成长因子
    revenue_cagr = calculate_cagr(pool['revenue'], 3)
    profit_cagr = calculate_cagr(pool['profit'], 3)
    momentum_20d = pool['close'].pct_change(20)

    # 3. 综合得分
    pool['score'] = (
        normalize(revenue_cagr) * 0.4 +
        normalize(profit_cagr) * 0.4 +
        normalize(momentum_20d) * 0.2
    )

    # 4. 选股
    selected = pool.nlargest(n_stocks, 'score')

    return selected
