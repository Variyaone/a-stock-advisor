# A股量化研究系统 - 专业版DAG

> 最终目标: 建设系统化、专业化的A股量化交易研究平台
> 目标基准: 比肩专业量化机构
> 核心要求: 最高阿尔法但不冒进
> 项目代号: a-stock-quant
> 启动时间: 2026-02-27 (18:12升级)

---

## 🎯 终极目标与标准

### 核心目标
**建设对标专业量化机构的A股量化研究平台，通过深度因子/指标/策略研究，追求最高阿尔法，提供每日交易建议**

**关键要求**:
- ✅ **方法论对标** - 与专业量化机构一致
- ✅ **深度研究** - 因子、指标、策略全链路研究
- ✅ **风险严控** - 不冒进，收益与风险平衡
- ✅ **每日输出** - 收盘后生成交易建议报告

### 核心约束
- ❌ A股无官方自动化API
- ✅ 方案: 每日收盘后生成交易建议报告

### 目标指标
| 指标 | 目标 |
|------|------|
| 年化超额收益 | >15% |
| 信息比率 | >1.5 |
| 最大回撤 | <25% |
| 行业偏离度 | <10% |
| 风格β | ≈0 |

## 📊 系统架构 (专业版)

```
                    +-------------------+
                    |   每日触发 (Cron)  |
                    |   收盘后 15:30    |
                    +---------+---------+
                              |
    +-------------------------------------------------------+
    |              A股量化研究系统 - 七层架构               |
    +-------------------------------------------------------+
    |                                                       |
    |  [数据层]          [因子层]         [策略层]          |
    |  ├─ 多源获取      ├─ 因子挖掘      ├─ 选股模型       |
    |  ├─ 清洗对齐      ├─ IC/IR检验     ├─ 择时逻辑       |
    |  └─ 存储管理      ├─ 正交化        └─ 仓位管理       |
    |                   └─ 因子组合                        |
    |                                                       |
    |  [组合层]          [回测层]         [风控层]          |
    |  ├─ 权重优化      ├─ 样本内外      ├─ 行业中性       |
    |  ├─ 中性化        ├─ 绩效分析      ├─ 风格中性       |
    |  └─ 再平衡        └─ 压力测试      ├─ 波动率控制     |
    |                                   └─ 风险限额       |
    |                                                       |
    |  [实盘层]                                              |
    |  ├─ 每日报告生成                                       |
    |  ├─ 飞书推送                                           |
    |  └─ 纸面交易跟踪                                       |
    |                                                       |
    +-------------------------------------------------------+
                              |
                    +---------v---------+
                    |   周期性优化      |
                    |   因子更新/调优   |
                    +-------------------+
```

### 各层核心要求

| 层级 | 专业做法 | 简易做法(❌) |
|------|----------|--------------|
| **数据层** | 多源交叉、清洗、对齐 | 单源直接用 |
| **因子层** | 挖掘+IC/IR/正交化 | 技术指标堆砌 |
| **策略层** | 多因子打分、机器学习 | 单指标策略 |
| **组合层** | 样本外优化、中性化 | 简单等权 |
| **回测层** | 样本内外、分年月 | 全样本无split |
| **风控层** | 行业/风格中性、VaR | 止损止盈 |
| **实盘层** | 监控、调优、跟踪 | 一次性报告 |
