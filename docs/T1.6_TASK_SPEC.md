## 实施步骤

### 步骤1: 计算因子相关性矩阵

```python
correlation_matrix = factors_df.corr(method='spearman')
```

#### 相关性标准
- |相关性| > 0.7：冗余，需合并/正交化
- |相关性| 0.5-0.7：相关但可保留
- |相关性| < 0.5：独立，保留

### 步骤3: 因子正交化

#### 方法A: 剩余化 (Residualization)
```python
# 对因子X在因子X1..Xn上回归，取残差
X_ortho = X - β1*X1 - β2*X2 - ... - βn*Xn
```

#### 方法B: 主成分分析 (PCA)
```python
from sklearn.decomposition import PCA
pca = PCA(n_components=k)
X_ortho = pca.fit_transform(X)
```

#### 建议
每组选1-2个代表因子，使用方法A正交化

### 步骤4: 最终因子选择评分

**综合评分公式**:
```python
factor_score = 0.4 * |IC_mean|_归一化 +
               0.3 * IR_归一化 +
               0.2 * 独立性得分 +
               0.1 * 互补性得分
```

#### 独立性得分
- 与其他因子平均相关性越低，得分越高

#### 互补性得分
- 不同类型因子(动量+价值+成长+情绪)组合更优

### 步骤5: 输出最终因子组合

#### 选择标准
1. **A级因子优先**: IC|>0.05, IR>1.0
2. **类型多样**: 至少包含3类因子
3. **相关性低**: |相关性|<0.5
4. **数量控制**: 5-10个因子
