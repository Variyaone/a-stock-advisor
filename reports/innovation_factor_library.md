## 📊 什么是创新因子？

创新因子是指通过学术研究、跨市场分析、另类数据挖掘等新颖方法发现的、尚未在主流因子库中广泛应用的量化因子。

### 创新因子来源

1. **学术前沿**：国际顶级金融期刊（JF、JFQA、RFS等）最新研究
2. **机构因子**：量化私募、公募基金的独门因子（公开论文/专利）
3. **跨市场因子**：从美股、港股等市场发现的有效因子应用于A股
4. **另类因子**：舆情情绪、北向资金、卖空数据、大宗交易等非常规数据

### 创新评价标准

**有效因子标准**：
- IC绝对值 > 0.02
- IR > 0.5
- IC胜率 > 55%
- 单调性 > 0.6（60%的分组满足单调性）

**创新质量分级**：
- 🟢 **A级（优秀）**：综合得分 >= 0.9，所有指标均通过
- 🟡 **B级（良好）**：综合得分 >= 0.7，至少2个指标通过
- 🟠 **C级（一般）**：综合得分 >= 0.5，至少1个指标通过
- 🔴 **D级（无效）**：综合得分 < 0.5

## 📈 创新因子记录

### 因子1：momentum_20d（20日动量因子）

**创建时间**: 2026-03-01
**类别**: 技术面因子 | 动量因子
**来源**: 经典文献验证

**因子公式**：
```
momentum_20 = (close_today - close_20_days_ago) / close_20_days_ago
```

**因子逻辑**：
- 衡量股票过去20天的价格动量
- 正向因子：动量越大，未来收益越高

**A股市效**：
- 动量效应在A股不如美股明显
- 反转效应在某些时期更强
- 需要与其他因子组合使用

**测试结果**：
- IC均值: 待测试
- IR: 待测试
- 单调性: 待测试

**备注**:
- 这是经典的动量因子，用于测试创新实验室功能
- 美股市场动量效应显著，A股需验证

### 因子2：volatility_20d_inv（20日低波动因子）

**创建时间**: 2026-03-01
**类别**: 技术面因子 | 波动率因子
**来源**: 低波动异常文献

**因子公式**：
```
return_1d = close_today / close_yesterday - 1
volatility_20 = std(return_1d over 20 days)
volatility_20_inv = 1 / volatility_20
```

**因子逻辑**：
- 衡量股票过去20天的波动率
- 低波动基金理论：低波动股票具有更高的风险调整后收益
- 因子值为波动率倒数，值越大表示越稳定

**A股市效**：
- 低波动异常在A股也存在
- 适合长期持有的投资者

**测试结果**：
- IC均值: 待测试
- IR: 待测试
- 单调性: 待测试

**备注**:
- "低波动之谜"是金融学经典异象
- 适合防御型组合
