# A股量化策略库

## 概述

本文档整理了A股市场中常用的量化策略，涵盖趋势跟踪、均值回归、因子轮动、行业轮动、事件驱动等多类策略。本策略库旨在建立全面的策略体系，为量化投资提供策略参考和组合思路。

---

## 一、趋势跟踪策略

### 1.1 策略概述

**核心逻辑**：追随市场趋势，"追涨杀跌"，上涨时买入，下跌时卖出。

**适用场景**：
- 单边上涨/下跌行情
- 趋势明显的市场
- 高波动市场

**优势**：
- 能够捕捉大级别行情
- 收益潜力大
- 实现简单

**劣势**：
- 震荡市表现差
- 容易错过局部高低点
- 可能产生假信号

### 1.2 典型策略

#### 1.2.1 双均线策略

**逻辑**：
```
买入信号：短期均线上穿长期均线（金叉）
卖出信号：短期均线下穿长期均线（死叉）
```

**参数**：
- 短期均线：5日、10日、20日
- 长期均线：30日、60日、120日

**变体**：
- 多均线系统（3条以上均线）
- 指数移动均线（EMA）
- 加权移动均线（WMA）

**A股表现**：
- 牛市表现优秀
- 熊市容易亏损
- 震荡市频繁止损

#### 1.2.2 布林带突破策略

**逻辑**：
```
上轨 = MA20 + 2*STD20
下轨 = MA20 - 2*STD20

买入信号：价格突破上轨
卖出信号：价格跌破下轨
```

**参数**：
- 中轨周期：20日
- 标准差倍数：2-3
- 也可用ATR替代标准差

**A股表现**：
- 捕捉突破行情效果好
- 突破假信号需过滤
- 结合成交量确认更可靠

#### 1.2.3 海龟交易法

**逻辑**：
```
唐奇安通道：
上轨 = 过去55日最高价
下轨 = 过去55日最低价

买入信号：价格超过55日高点
卖出信号：价格跌破20日低点
```

**参数**：
- 入场：55日突破
- 止损：2日平均真实波幅（ATR）
- 止盈：55日突破或10日低点
- 仓位管理：单位风险头寸

**A股表现**：
- 长期趋势有效
- 需要严格纪律
- 止损至关重要

#### 1.2.4 自适应趋势跟踪

**逻辑**：
使用动态调整的趋势识别方法，如Kama（Kaufman Adaptive Moving Average）、自适应均线等。

**优点**：
- 自适应调整速度
- 震荡市减仓
- 趋势市加速

**实现**：
```
效率比率 = |方向变动| / |总波动|
Kama = Kama_prev + SC * (Price - Kama_prev)
SC = [效率比率 * (最快SC - 最慢SC) + 最慢SC]^2
```

### 1.3 优化方向

| 优化方法 | 说明 |
|---------|------|
| **趋势确认** | 使用成交量、换手率确认突破有效性 |
| **过滤假信号** | 结合振荡指标（RSI、KDJ）过滤 |
| **动态参数** | 根据市场波动率调整均线周期 |
| **仓位管理** | 趋势强度动态调整仓位 |
| **多时间周期** | 大周期趋势+小周期入场 |

## 二、均值回归策略

### 2.1 策略概述

**核心逻辑**：价格长期围绕均值波动，偏离均值过度会回归。"低买高卖"。

**适用场景**：
- 震荡市
- 价格高波动
- 回归明显的市场

**优势**：
- 震荡市表现好
- 捕捉超卖超买机会
- 风险相对可控

**劣势**：
- 趋势市容易亏损
- 反弹时机难把握
- 可能被套在下跌趋势中

### 2.2 典型策略

#### 2.2.1 RSI均值回归

**逻辑**：
```
RSI = 100 - 100 / (1 + RS)

买入信号：RSI < 30（超卖）
卖出信号：RSI > 70（超买）

A股特点：RSI < 25更可靠，RSI > 75要谨慎
```

**参数**：
- 周期：14日（标准）、7日（灵敏）、21日（稳健）
- 超买线：70
- 超卖线：30

**A股表现**：
- 反转效应比美股更明显
- 2017年后反转效应减弱
- 需要结合市值、流动性格外外

#### 2.2.2 MACD反转

**逻辑**：
```
MACD = EMA12 - EMA26
Signal = EMA9(MACD)
Histogram = MACD - Signal

买入信号：MACD金叉+柱状图翻正
卖出信号：MACD死叉+柱状图翻负
```

**参数**：
- 快速EMA：12日
- 慢速EMA：26日
- 信号EMA：9日

**A股表现**：
- 捕捉中短期反转效果好
- 需要结合零轴判断趋势
- 背离信号更重要

#### 2.2.3 布林带均值回归

**逻辑**：
```
买入信号：价格跌破下轨（可能超卖反弹）
卖出信号：价格突破上轨（可能回调）

关键：等待价格回到中轨再确认
```

**参数**：
- 中轨：20日均线
- 标准差：2
- 可根据波动率调整

**A股表现**：
- 宽幅震荡效果最好
- 单边趋势容易亏损
- 结合成交量更可靠

#### 2.2.4 统计套利

**逻辑**：
```
价差 = 价格A - 价格B（或协整关系）

策略：
- 当价差偏离均值+2σ时，做空价差
- 当价差回归均值时平仓

配对选择：
- 行业相似
- 市值相近
- 历史协整关系稳定
```

**A股应用**：
- 同行业龙头配对
- A+H股套利
- ETF与成分股套利

#### 2.2.5 偏离度策略

**逻辑**：
```
偏离度 = (当前价格 - MA20) / MA20

买入信号：偏离度 < -10%（过度下跌）
卖出信号：偏离度 > +10%（过度上涨）

可动态调整阈值：
    阈值 = k * σ(偏离度历史)
```

**A股特点**：
- 小盘股偏离度阈值更大
- 高成长板块偏离容忍度更高
- 市场整体情绪影响阈值

### 2.3 优化方向

| 优化方法 | 说明 |
|---------|------|
| **动态阈值** | 根据波动率调整买卖阈值 |
| **多指标组合** | RSI+MACD+KDJ多重确认 |
| **分批建仓** | 价格越偏离加仓越多 |
| **止损机制** | 趋势确认后及时止损 |
| **板块轮动** | 配对板块均值回归 |

## 三、因子轮动策略

### 3.1 策略概述

**核心逻辑**：不同因子在不同市场环境下表现不同，根据因子表现动态调整因子权重。

**适用场景**：
- 市场风格切换频繁
- 因子收益率不稳定
- 需要动态配置的场景

**优势**：
- 适应市场变化
- 降低因子风险
- 提高整体收益

**劣势**：
- 因子切换成本高
- 可能误判风格
- 实施复杂度高

### 3.2 典型策略

#### 3.2.1 基于因子IC的轮动

**逻辑**：
```
1. 每周计算各因子的滚动IC（如12周滚动）
2. 选择IC最高的Top-N因子
3. 调整因子权重（IC加权或等权）
4. 构建多因子组合
```

**参数**：
- 滚动窗口：12周（1季度）
- 因子数量：3-5个
- 调仓频率：每周或每月

**A股表现**：
- 估值、质量因子长期稳定
- 动量因子周期性表现
- 需要避免过度频繁调仓

#### 3.2.2 基于市场风格的轮动

**市场风格分类**：
- 成长风格 vs 价值风格
- 大盘 vs 小盘
- 动量 vs 反转
- 高贝塔 vs 低波动

**轮动逻辑**：
```
1. 判断当前市场风格（宏观、估值、情绪）
2. 匹配对应的因子组合：
   - 价值风格：PE/PB低因子+质量因子
   - 成长风格：营收/盈利增长因子+动量因子
   - 风险偏好高：小盘因子+高动量
   - 风险厌恶：大盘因子+低波动
3. 调整因子权重
```

**判断因子**：
- **宏观**：GDP、利率、流动性
- **估值**：市场PE/PB水平
- **情绪**：融资余额、换手率
- **技术**：均线系统、趋势指标

#### 3.2.3 基于动量的因子轮动

**逻辑**：
```
1. 计算各因子组合的过去收益率（如3个月）
2. 选择收益率最高的组合
3. 动量持续则持有，动量衰减则切换
```

**参数**：
- 观察期：3-6个月
- 切换阈值：收益率排名变化>20%
- 止损：组合回撤>10%切换

**A股表现**：
- 因子动量持续时间较短（1-3个月）
- 需要快速反应
- 适合高换手率的机构

#### 3.2.4 基于风险平价的因子轮动

**逻辑**：
```
1. 计算各因子的风险（标准差）
2. 追求各因子风险贡献相等
3. 权重 = 目标风险 / 因子风险
```

**数学表达**：
```
w_i ∝ 1 / σ_i

等风险：w_i * σ_i = constant（所有i）
```

**优点**：
- 风险分散
- 稳健性强
- 抗黑天鹅

**A股应用**：
- 适合长期配置型资金
- 结合风格中性

### 3.3 因子组合示例

**稳健组合（适合长期持有）**：
```
- 估值因子（30%）：PE_TTM行业中性
- 质量因子（25%）：ROE行业中性
- 低波动因子（20%）：20日波动率倒数
- 规模因子（15%）：市值偏中小盘
- 动量因子（10%）：60日动量
```

**成长组合（适合牛市）**：
```
- 成长因子（40%）：营收增长+盈利增长
- 动量因子（30%）：120日动量
- 分析师预期（20%）：目标价涨幅
- 规模因子（10%）：中等市值
```

**防守组合（适合熊市）**：
```
- 低波动因子（40%）：20日波动率倒数
- 现金流因子（30%）：经营现金流/净利润
- 高股息因子（20%）：股息率
- 价值因子（10%）：PB行业中性
```

## 四、行业轮动策略

### 4.1 策略概述

**核心逻辑**：不同行业在不同经济周期阶段表现不同，根据宏观经济形势动态调整行业配置。

**适用场景**：
- 宏观经济周期明显
- 行业分化严重
- 政策驱动市场

**优势**：
- 捕捉宏观机会
- 提高组合弹性
- 分散行业风险

**劣势**：
- 宏观判断难度大
- 政策影响不可预测
- 行业切换成本高

### 4.2 宏观经济周期与行业配置

#### 4.2.1 美林时钟

| 阶段 | 宏观特征 | 经济特征 | 推荐行业 |
|-----|---------|---------|---------|
| **复苏期** | GDP↑，通胀↓ | 经济复苏，需求恢复 | 周期品（有色、化工）、银行、可选消费 |
| **过热期** | GDP↑，通胀↑ | 经济过热，需求旺盛 | 能源、大宗商品、原材料 |
| **滞胀期** | GDP↓，通胀↑ | 经济放缓，通胀高企 | 必选消费、公用事业、黄金 |
| **衰退期** | GDP↓，通胀↓ | 经济衰退，需求萎缩 | 成长股、医药、国防军工 |

#### 4.2.2 A股市场特点

**政策驱动强**：
- 政策利好：新能源、半导体、军工
- 政策利空：地产、教培、互联网

**资金轮动**：
- 市场资金有限，板块跷跷板效应
- 北向资金偏好大盘蓝筹
- 游资偏好题材炒作

**估值波动大**：
- 行业估值周期性变化明显
- 需结合估值位置决策

### 4.3 典型策略

#### 4.3.1 基于行业动量的轮动

**逻辑**：
```
1. 计算各行业指数（或ETF）的过去收益率
2. 选择收益率最高的Top-N行业
3. 构建"动量组合"
4. 定期调仓（周度/月度）
```

**参数**：
- 观察期：1个月、3个月、6个月
- 行业数量：3-5个
- 调仓频率：每周或每月

**A股表现**：
- 短期动量（1个月）效果最好
- 中期动量（3-6个月）效果一般
- 需要考虑流动性

#### 4.3.2 基于行业相对估值的轮动

**逻辑**：
```
1. 计算各行业PE/PB的历史分位数
2. 选择估值分位数最低的行业（相对低估）
3. 结合基本面（营收、盈利）确认反转信号
4. 逐步建仓
```

**参数**：
- 估值分位数：<25%（低估）、>75%（高估）
- 基本面确认：营收增长>0%、ROE>行业平均
- 仓位控制：逐步加仓

**A股表现**：
- 低估值策略长期有效
- 需要耐心等待反转
- 避免价值陷阱

#### 4.3.3 基于宏观因子的轮动

**逻辑**：
```
1. 建立宏观因子库：
   - 利率：10年期国债收益率
   - 通胀：CPI、PPI
   - 流动性：社融M2
   - 汇率：美元兑人民币
   - PMI：制造业PMI

2. 回测各行业对宏观因子的暴露：
   - 行业收益率 = α + β1*利率 + β2*通胀 + β3*流动性 + ...

3. 预测宏观因子走势：
   - 分析央行政策
   - 研究经济数据趋势

4. 选择对有利宏观因子暴露高的行业
```

**适用行业**：
- 利率敏感：银行、地产、公用事业
- 通胀受益：能源、资源、农业
- 流动性受益：券商、成长股
- 人民币升值受益：航空、造纸

#### 4.3.4 基于资金流向的轮动

**逻辑**：
```
1. 跟踪各行业资金流向：
   - 北向资金：陆股通持仓
   - 机构资金：基金重仓股
   - 游资：龙虎榜

2. 分析资金流向趋势：
   - 净流入持续 → 可能有机会
   - 净流出持续 → 风险较大

3. 均值回归策略：
   - 资金过度流入后流出 → 卖出
   - 资金过度流出后回流 → 买入
```

**数据来源**：
- 东方财富、同花顺资金流向数据
- Wind、Choice资管数据
- 上交所、深交所Level-2数据

#### 4.3.5 基于事件驱动的轮动

**常见事件类型**：

**政策事件**：
- 产业政策支持 → 受益板块
- 政策收紧 → 受损板块

**财报事件**：
- 行业整体景气变化
- 龙头公司财报超预期

**外部事件**：
- 新能源产业：碳中和政策
- 半导体：国产替代
- 军工：地缘政治
- 医药：集采政策

**操作流程**：
```
1. 监测事件（政策、财报、新闻）
2. 评估事件对行业的影响
3. 判断市场反应是否充分
4. 反应不足 → 买入
5. 反应过度 → 卖出或做空
```

### 4.4 行业配置框架

**自上而下配置流程**：
```
1. 宏观判断
   ├─ 经济周期阶段
   ├─ 货币政策方向
   └─ 流动性环境

2. 风格判断
   ├─ 成长 vs 价值
   ├─ 大盘 vs 小盘
   └─ 动量 vs 反转

3. 行业选择
   ├─ 顺周期/逆周期
   ├─ 政策受益/受损
   └─ 估值位置

4. 个股选择
   ├─ 行业龙头
   ├─ 业绩确定性高
   └─ 流动性好

5. 风险管理
   ├─ 行业分散
   ├─ 及时止损
   └─ 动态调整
```

**风险控制**：
- 单行业仓位<30%
- 相关性高的行业对冲
- 及时止损（-15%）
- 动态监控

## 五、事件驱动策略

### 5.1 策略概述

**核心逻辑**：跟踪特定事件（财报发布、分红派息、并购重组等），利用市场对事件反应不足或过度反应获取超额收益。

**适用场景**：
- 事件发布
- 政策变化
- 宏观事件

**优势**：
- 收益确定性强
- 事件可预先知晓
- 风险相对可控

**劣势**：
- 机会有限
- 信息不对称
- 流动性风险

### 5.2 典型策略

#### 5.2.1 财报超预期策略

**逻辑**：
```
1. 公司发布财报
2. 实际业绩 > 市场预期 → 超预期
3. 买入超预期股票
4. 持有直到效应减弱（通常1-3个月）
```

**衡量超预期**：
- 实际EPS vs 分析师预期
- 实际营收 vs 分析师预期
- 实际ROE vs 历史平均

**A股表现**：
- 超预期效应持续1-2个月
- 中小盘股效应更明显
- 需要过滤炒作概念股

**优化方向**：
- 结合业绩预告
- 看分析师调整
- 看业绩说明会

#### 5.2.2 分红派息策略

**除息前**：
```
逻辑：临近除息，股价通常上涨
操作：除息前1-2周买入
风险：除息后可能下跌
```

**除息后**：
```
逻辑：除息后股价下跌，可能被低估
操作：除息后买入
适用：高股息、ROE稳定的公司
```

**A股特点**：
- 分红季（6-7月）效应明显
- 高股息股适合长期持有
- 避免分红造假的公司

#### 5.2.3 兼并收购（M&A）

**逻辑**：
```
1. 挖掘"潜在被收购标的"：
   - 估值低（PB<1.5）
   - 市值小（<100亿）
   - 现金多（现金/市值>0.5）
   - 业务与战略投资者契合

2. "套利交易"：
   - 收购公告后
   - 买入标的股票（如果当前价<收购价）
   - 等待交易完成

3. "抢帽子"：
   - 提前预测可能并购
   - 提前埋伏
```

**风险**：
- 交易风险（失败）
- 时间成本
- 流动性风险

#### 5.2.4 定增策略

**逻辑**：
```
1. 关注定增项目：
   - 融资用途（扩产、并购）
   - 认购对象（大股东知名机构）
   - 折价率（>10%）

2. 参与定增或：
   - 公告日买入
   - 解禁日前卖出

3. 定增解禁策略：
   - 解禁前股价下跌 → 可能是机会
   - 解禁后股价上涨 → 可能是机会
```

**A股表现**：
- 定增公告日效应明显
- 解禁日效应分化
- 需要判断定增质量

#### 5.2.5 业绩预告策略

**逻辑**：
```
1. 关注业绩预告期：
   - 半年报：7月
   - 年报：1月

2. 选择预告业绩大增的公司：
   - 净利润增长>50%（大幅预增）
   - 扭亏为盈
   - 业绩超预期

3. 提前布局：
   - 预告发布前可能有泄露
   - 公告后追涨
```

**A股表现**：
- A股业绩预告效应较强
- 需要区分"业绩反转"和"业绩造假"
- 结合行业景气度

#### 5.2.6 政策事件驱动

**典型政策事件**：

**产业政策**：
```
新能源政策 → 光伏、风电、新能源汽车
半导体国产化 → 半导体设备、材料
碳中和 → 环保、新能源
```

**货币政策**：
```
降准降息 → 银行、地产、券商
紧缩政策 → 防守型行业
```

**财政政策**：
```
基建投资 → 建筑、建材、工程机械
税收优惠 → 受益行业
```

**区域政策**：
```
自贸区 → 区域内的上市公司
雄安新区 → 河北本地股
粤港澳大湾区 → 广东、香港、澳门股
```

**操作流程**：
```
1. 监测政策动向（新闻、公告、会议）
2. 分析政策影响（直接/间接、短期/长期）
3. 选择受益股：
   - 行业龙头
   - 弹性大
   - 估值合理
4. 及时建仓：
   - 公告日第一时间反应
   - 注意追高风险
5. 见好就收：
   - 等待政策落地
   - 避免预期落空
```

#### 5.2.7 特殊事件驱动

**停复牌**：
```
长期停牌股票：
- 正常复牌 → 可能补涨/补跌
- 被重组 → 可能暴涨
```

**ST摘帽**：
```
ST公司改善经营 → 可能摘帽 → 股价上涨
```

**高管增持**：
```
内部人增持 → 释放信心信号 → 股价可能上涨
```

**北向资金增持**：
```
北向资金流入 → 外资看好 → 股价可能上涨
```

### 5.3 事件驱动策略框架

**事件监控**：
```
├── 宏观事件
│   ├── 央行政策（降准降息、MLF）
│   ├── 财政政策（财税、基建）
│   └── 监管政策（规范、限制）
│
├── 微观事件
│   ├── 财报发布（年报、季报）
│   ├── 业绩预告
│   ├── 分红派息
│   ├── 高管增持减持
│   ├── 并购重组
│   └── 定增解禁
│
├── 行业事件
│   ├── 产业政策
│   ├── 技术突破
│   └── 供给冲击（涨价、减产）
│
└── 外部事件
    ├── 地缘政治
    ├── 全球经济
    └── 国际关系
```

**事件评估**：
```
1. 重要性
   ├─ 影响范围（全市场/行业/个股）
   ├─ 影响时间（短期/长期）
   └─ 影响方向（利好/利空）

2. 确定性
   ├─ 已确定（正式公告）
   ├─ 高概率（预期内）
   └─ 低概率（不确定）

3. 市场反应
   ├─ 是否充分定价
   ├─ 是否过度反应
   └─ 预期差是否存在

4. 可操作性
   ├─ 流动性
   ├─ 估值安全边际
   └─ 仓位大小
```

## 六、混合策略与组合优化

### 6.1 多策略组合

**基本思路**：将不同类型的策略组合，分散风险，提高收益。

**常见组合**：

**趋势+均值回归**：
```
趋势策略权重：40%
├─ 上涨趋势 → 趋势策略为主
└─ 震荡市场 → 均值回归为主

均值回归权重：40%
├─ 趋势策略失效时提供缓冲
└─ 捕捉背离机会

其他策略：20%
├─ 事件驱动
└─ 因子轮动
```

**A股特性组合**：
```
价值风格（35%）：低估值、质量因子
成长风格（25%）：成长因子、动量因子
风格轮动（20%）：根据风格切换调整
事件驱动（15%）：财报、政策事件
行业轮动（5%）：行业景气配置
```

### 6.2 风险平价组合

**理论**：
```
各策略风险贡献相等
w_i * σ_i = constant（所有策略i）
```

**优势**：
- 风险分散
- 抗市场波动
- 适合长期配置

**缺点**：
- 可能高杠杆
- 收益可能不如激进组合
- 需要动态调整

### 6.3 动态权重调整

**根据市场环境调整**：
```
牛市增长期：
├─ 趋势跟踪 → 40%
├─ 动量策略 → 30%
└─ 价值策略 → 30%

熊市下跌期：
├─ 防守策略 → 40%
├─ 低波动 → 30%
└─ 现金等价物 → 30%

震荡市：
├─ 均值回归 → 40%
├─ 低频交易 → 30%
└─ 事件驱动 → 30%
```

**调整触发条件**：
- 市场趋势识别（MA20/60/120）
- 波动率突变（20日波动率）
- 风格切换（成长/价值轮动）

## 七、策略评估与风险管理

### 7.1 策略评估指标

**收益指标**：
- 年化收益率
- 累计收益率
- 最大回撤
- 胜率

**风险调整指标**：
- 夏普比率 > 1.0为优秀
- 卡玛比率 > 0.5为优秀
- 信息比率 > 0.5为优秀
- 索提诺比率

**交易指标**：
- 年化换手率
- 平均持仓天数
- 单笔交易收益
- 交易滑点

**A股特殊性**：
- 考虑涨跌停限制
- 考虑T+1交易规则
- 考虑印花税（-0.15%）和佣金（-0.0325%）

### 7.2 风险管理框架

**仓位控制**：
```
单策略仓位：≤总资金的30%
单股票仓位：≤总资金的5%
相关行业对冲：相关性>0.7的行业对冲
```

**止损机制**：
```
策略级止损：单策略回撤>15%减仓
股票级止损：单股回撤>20%止损
市场级止损：市场下跌>10%整体减仓
```

**动态监控**：
```
实时监控：
├─ 策略净值
├─ 风险暴露
├─ 相关性变化
└─ 异常交易
```

**压力测试**：
```
定期情景测试：
├─ 2008年式大跌（-50%）
├─ 2015年式股灾（-40%）
├─ 2020年式疫情（-30%）
└─ 2024年式震荡

评估策略在极端情况下的表现：
├─ 最大回撤
├─ 恢复周期
└─ 暴雷概率
```

### 7.3 实施注意事项

**市场环境变化**：
- A股市场结构变化（散户→机构）
- 量化交易增多，策略拥挤
- 监管政策变化

**技术实现**：
- 低延迟交易硬件
- 稳定的数据源
- 回测框架（Qlib、Backtrader）

**合规性**：
- 避免内幕交易
- 符合监管要求
- 信息披露

## 九、结论

A股量化策略选择需要：
1. **匹配自身资源**：资金规模、技术水平、风险偏好
2. **匹配市场环境**：牛市/熊市/震荡市
3. **风险可控**：分散、止损、动态调整
4. **持续优化**：策略演进、参数调优

**建议**：
- 从简单策略开始（多因子选股）
- 逐步加入复杂策略（事件+轮动）
- 始终进行回测和小仓验证
- 关注策略拥挤度变化
- 保持学习和适应
