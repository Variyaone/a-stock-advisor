# A股量化因子库

## 概述

本文档整理了A股量化投资中常用的因子，覆盖基本面、技术面、情绪和另类数据等多个维度，共计80+个因子。本因子库的目标是建立全面的因子体系，支持多因子选股和量化策略开发。

## 因子分类体系

A类因子按数据来源可分为四大类：
1. **基本面因子** - 基于财务报表数据构建
2. **技术面因子** - 基于价量数据构建
3. **情绪因子** - 基于市场情绪和舆情数据
4. **另类数据因子** - 基于分析师预期、机构持仓等外部数据

## 一、基本面因子（32个因子）

### 1.1 估值因子（10个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| PE（市盈率） | 总市值 / 净利润 | 低估值公司可能有投资价值 |
| PB（市净率） | 总市值 / 净资产 | 衡量股价相对于净资产的溢价程度 |
| PS（市销率） | 总市值 / 营业收入 | 适用于尚未盈利的成长股估值 |
| PCF（市现率） | 总市值 / 经营现金流 | 现金流比利润更难做假，更真实 |
| PEG（市盈增长率） | PE / 营业收入增长率 | 综合考虑估值与成长性，PEG<1为低估 |
| EV/EBITDA | 企业价值 / EBITDA | 剔除资本结构影响，更适合横向对比 |
| 企业价值/自由现金流 | EV / FCFF | 衡量企业创造自由现金流的能力 |
| 市值 | 总市值 | A股存在小盘效应，小盘股可能有超额收益 |
| 股息率 | 年度股息 / 股价 | 高股息股票可能有稳定回报 |
| 市盈率倒数 | 1 / PE | 可以直接理解为收益率 |

### 1.2 盈利能力因子（7个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| ROE（净资产收益率） | 净利润 / 净资产 | 衡量股东投入资本的盈利能力，巴菲特最看重的指标 |
| ROA（总资产收益率） | 净利润 / 总资产 | 衡量企业整体资产的盈利能力 |
| ROIC（投入资本收益率） | NOPAT / 投入资本 | 更准确地衡量资本使用效率 |
| 毛利率 | (营业收入-营业成本)/营业收入 | 衡量产品定价能力和成本控制能力 |
| 净利率 | 净利润 / 营业收入 | 衡量企业最终盈利能力 |
| 销售现金比率 | 经营现金流 / 营业收入 | 衡量营收转化为现金的能力 |
| 资产周转率 | 营业收入 / 平均总资产 | 衡量资产使用效率 |

### 1.3 成长性因子（5个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 营收增长率 | (本期营收-上期营收)/上期营收 | 衡量企业规模扩张速度 |
| 净利润增长率 | (本期净利-上期净利)/上期净利 | 衡量盈利增长能力 |
| ROE增长率 | (本期ROE-上期ROE)/上期ROE | 衡量股东权益回报的提升速度 |
| 毛利率变化率 | 本期毛利率-上期毛利率 | 衡量盈利质量的改善/恶化 |
| 总资产增长率 | (本期资产-上期资产)/上期资产 | 衡量企业成长速度 |

### 1.4 偿债能力因子（5个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 资产负债率 | 总负债 / 总资产 | 衡量企业财务杠杆水平，过高有风险 |
| 流动比率 | 流动资产 / 流动负债 | 衡量短期偿债能力，一般认为>2较安全 |
| 速动比率 | (流动资产-存货)/流动负债 | 剔除存货影响，更严格的短期偿债能力指标 |
| 产权比率 | 总负债 / 股东权益 | 衡量债权人资金与股东资金的比率 |
| 利息保障倍数 | EBIT / 利息费用 | 衡量偿付利息的能力，倍数越高越安全 |

### 1.5 营运能力因子（5个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 存货周转率 | 营业成本 / 平均存货余额 | 衡量存货管理效率，周转越快越好 |
| 应收账款周转率 | 营业收入 / 平均应收账款 | 衡量应收账款回收速度 |
| 总资产周转率 | 营业收入 / 平均总资产 | 衡量资产利用效率 |
| 现金转化周期 | 存货周转天数+应收周转天数-应付周转天数 | 衡量现金被占用的时间 |
| 固定资产周转率 | 营业收入 / 平均固定资产 | 衡量固定资产使用效率 |

## 二、技术面因子（30个因子）

### 2.1 动量因子（10个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 20日动量 | (今日收盘价-20日前收盘价)/20日前收盘价 | 短期价格趋势，强者恒强 |
| 60日动量 | (今日收盘价-60日前收盘价)/60日前收盘价 | 中期价格趋势 |
| 120日动量 | (今日收盘价-120日前收盘价)/120日前收盘价 | 长期价格趋势 |
| 月度动量 | 本月收益率 | 跨月比较 |
| 季度动量 | 季度收益率 | 跨季比较 |
| 加权20日动量 | Σ(每日收益率×权重) | 近期权重更大的动量 |
| 换手率加权动量 | 换手率加权平均收益率 | 结合活跃度的动量 |
| 指数衰减动量 | 指数衰减加权历史收益率 | 灵敏度更高的动量 |
| 动量加速度 | 当前动量-前期动量 | 衡量动量变化速度 |
| 相对强度 | 个股收益率/市场收益率 | 相对市场的表现 |

### 2.2 反转因子（6个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 5日反转 | -5日收益率 | 超跌可能反弹（A股反转效应明显） |
| 20日反转 | -20日收益率 | 中期反转 |
| 60日反转 | -60日收益率 | 长期反转 |
| 偏离度 | (当前收盘价-MA20)/MA20 | 衡量相对均线的偏离程度，过度偏离可能回归 |
| RSI反转 | 100-RSI（或RSI） | RSI超买超卖信号 |
| 威廉指标反转 | -%R | %R超买超卖信号 |

### 2.3 波动率因子（7个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 历史波动率（20日） | 过去20日收益率标准差 | 衡量价格波动风险 |
| 历史波动率（60日） | 过去60日收益率标准差 | 长期波动率 |
| 已实现波动率 | 日内收益率波动率 | 更精确的波动率衡量 |
| ATR（平均真实波幅） | ATR指标 | 衡量每日价格波动幅度 |
| 上行波动率 | 只上涨日收益率标准差 | 衡量上涨时的波动 |
| 下行波动率 | 只下跌日收益率标准差 | 衡量下跌时的波动 |
| 波动率偏度 | 上行波动率/下行波动率 | 衡量涨跌不对称性 |

### 2.4 流动性因子（7个）

| 因子名称 | 计算公式 | 因子逻辑 |
|---------|---------|---------|
| 换手率 | 成交量 / 流通股本 | 衡量股票活跃度，换手率过高可能有风险 |
| Amihud非流动性 | ｜日收益率｜/成交额 | 衡量交易对价格的影响，值越大流动性越差 |
| 流通市值 | 流通股数×股价 | 衡量可交易规模 |
| 买卖价差 | (卖价-买价)/中间价 | 衡量交易成本 |
| 成交额占市值比 | 日成交额/流通市值 | 衡量流动性充足程度 |
| 换手率波动率 | 换手率的标准差 | 衡量流动性稳定性 |
| 零成交日占比 | 零成交天数/总天数 | 高零成交占比表示流动性差 |

## 五、因子使用建议

### 5.1 因子有效性

根据历史研究，A股市场中以下因子较为有效：

**高IC因子**：
- 规模因子（小盘效应）
- 估值因子（低估值效应）
- 动量因子（美股动量，A股反转）
- 波动率因子（低波动效应）
- 质量因子（高ROE、高ROA）

**因子特点**：
- **美股**：动量效应明显
- **A股**：反转效应比动量更显著，尤其是2017年后
- **港股**：介于两者之间

### 5.2 因子选择策略

1. **相关性控制**：选择相关性低于0.6的因子组合
2. **IC稳定性**：选择IC均值>3%，IR>0.5的因子
3. **容量考虑**：考虑因子容量，避免过度拥挤
4. **换手率控制**：平衡换手率与交易成本
5. **风格中性**：可考虑对规模、价值等风格因子进行中性化

### 5.3 因子组合示例

**保守组合**：
- 估值因子(30%)：PE低、PB低
- 质量因子(30%)：ROE高、ROA高
- 波动率因子(20%)：低波动
- 规模因子(20%)：中小盘

**成长组合**：
- 成长因子(40%)：营收增长、盈利增长
- 动量因子(30%)：中期动量
- 分析师预期(20%)：目标价涨幅高
- 市值因子(10%)：中等市值

**多因子组合**：
- 基本面因子(40%)：估值+质量+成长
- 技术面因子(30%)：动量+反转+低波
- 情绪因子(15%)：分析师情绪+舆情
- 另类因子(15%)：机构持仓+北向资金

## 六、因子数据来源

### 6.1 基本面数据
- 来源：财务报表（利润表、资产负债表、现金流表）、公告
- 更新频率：季度/半年度/年度
- 可靠性：高（审计后）

### 6.2 技术面数据
- 来源：交易所行情数据
- 更新频率：日度/分钟级别
- 可靠性：极高

### 6.3 情绪数据
- 来源：媒体、社交平台、搜索数据
- 更新频率：日度/实时
- 可靠性：中等（需要清洗）

### 6.4 另类数据
- 来源：研报、机构公告、交易所
- 更新频率：不定期
- 可靠性：中高

## 七、因子挖掘方法

### 7.1 人工挖掘
- 基于金融理论推导
- 行业经验总结
- 文献研究

### 7.2 自动化挖掘
- **遗传规划**：使用进化算法搜索因子表达式
- **OpenFE**：特征工程自动化工具
- **RD-Agent(Q)**：多智能体框架自动挖掘
- **深度强化学习**：训练智能体生成因子
- **遗传算法**：交叉变异筛选因子

### 7.3 因子评估指标
- **IC（Information Coefficient）**：因子与未来收益率的相关系数
- **IR（Information Ratio）**：IC均值/IC标准差，衡量稳定性
- **Rank IC**：排序相关性
- **换手率**：因子组合平均换手率
- **容量**：可容纳的资金规模

## 八、注意事项

1. **过拟合风险**：避免样本内过度优化，需样本外验证
2. **因子失效**：因子有效性会衰减，需持续监控
3. **交易成本**：高换手率因子需考虑交易成本
4. **风格暴露**：注意组合的风格暴露风险
5. **数据质量**：确保数据准确、完整、一致
6. **市场变化**：A股市场结构变化会影响因子表现
