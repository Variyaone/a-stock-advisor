## 🎯 如何使用

### 快速开始

#### 1. 配置飞书Webhook（必需）
```bash
cd /Users/variya/.openclaw/workspace/projects/a-stock-advisor

# 复制配置模板
cp config/.sample/feishu_config.json config/feishu_config.json

# 编辑配置文件，填入你的飞书Webhook URL
vim config/feishu_config.json
```

配置内容：
```json
{
  "webhook_url": "https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx",
  "enabled": true,
  "push_time": "18:35",
  "timezone": "Asia/Shanghai"
}
```

#### 2. 手动运行测试
```bash
# 运行每日选股
python3 run_daily.py

# 或使用测试脚本
./test_automation.sh
```

#### 3. 安装定时任务
```bash
# 自动安装（推荐）
./install_cron.sh

# 系统会每天18:30自动运行
# 查看日志：tail -f logs/daily_run.log
```

### 日常使用

**自动运行**:

- 周末和节假日自动跳过

## ⚠️ 重要说明

### 1. 风险提示
- **本系统仅供研究参考，不构成投资建议**
- 实盘效果可能与回测存在较大差异
- 现有策略收益较低（年化5.78%），需优化
- 股票样本量较小（54只），可能过拟合

### 2. 已知限制
- 交易成本未考虑（大约会降低1-2%收益）
- 行业集中度较高（未实现真正的行业中性）
- 有效因子仅1个，稳健性不足
- 风险控制目前是警告机制，未实际执行止盈止损

### 3. 数据来源
- 当前数据：模拟数据（2021-2025）
- 更新频率：尚未配置自动化更新
